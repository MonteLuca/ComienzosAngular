Para desplegar la carpeta dist se necesita el siguiente comando 
ng build --configuration production.

Para crear un modulo se utiliza el siguiente comando
ng generate module shared
o de forma mas abreviada
ng g m shared


Para crear un componente se utiliza el siguiente comando
ng generate component "nombre de la carpeta"/ sidebar
o de forma mas abreviada 
ng g c "nombre de la carpeta"/ sidebar
